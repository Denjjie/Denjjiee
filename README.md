##  Projects

###  Panabo Voting System
> *IT112 – Computer Programming 1*

A Java-based console application designed to simplify and secure the voting process for Panabo City. It replaces traditional paper-based methods with a focused, self-contained election workflow.

**Key Features:**
- Voter registration with Barangay ID, full name, and age validation
- Dual election support — Municipal Election (18+) and SK Election (ages 15–30)
- Double-voting prevention per registered user
- Admin dashboard to start/end elections and manage accounts
- Post-election consolidated results viewable by all users

**Tech:** Java (Console Application)

###  S.A.K.A. – Smart Advisory for Klima-resilient Agriculture
> *ITELEC2 – Project Proposal*

A web-based advisory platform designed to empower Filipino farmers against climate change through data-driven insights and community collaboration.

**Key Features:**
- Localized weather forecasts tailored to farming activities
- Adaptive crop calendars based on real-time climate conditions
- Irrigation planning tools for optimized water use
- Pest and disease management advisories
- Community forum for farmers to share tips and innovations
- Admin analytics dashboard for usage monitoring
- Knowledge base of climate-smart farming practices

**Tech:** Web Application (Proposed) | Focus: Usability, Accessibility, Localization for PH context
